# IG-Mapas_y_Terrenos
El csv utilizado tiene las coordenadas de latitud y longitud de las diferentes centrales energéticas de españa. Además de estos datos el trabajo hace uso de la columna Combustible, para clasificar los diferentes tipos de centrales y representarlos en el mapa con una modelo 3D acorde a sus tipo de combustible, el año en el comisionó la central, se usará para a través de un slider mostrar o no la central en el mapa, y la generación de gigavatios hora que se estima que genererá la central, este dato se mostrará mediante una barra amarilla saliendo de la central.
